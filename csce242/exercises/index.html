CSCE 242: Web Applications

Christopher Wingo

In this class, you'll learn to build and style a website with HTML and CSS. You'll add interactive features with JavaScript and use Node.js to handle data on a server.

Assignments

Assignment 1 – Basic HTML

This assignment covers the fundamentals of HTML, including the structure of a webpage and essential tags.

Assignment 2 – Basic CSS

This assignment focuses on styling a webpage using CSS. You'll learn about selectors, properties, and how to create appealing layouts.

Assignment 3 – Page Layout

In this assignment, you'll learn how to create responsive page layouts using Flexbox and media queries.
