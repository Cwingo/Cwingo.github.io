CSCE 242: Web Applications

Christopher Wingo

In this class, you'll learn to build and style a website with HTML and CSS. You'll add interactive features with JavaScript and use Node.js to handle data on a server.

You'll learn about selectors, properties, and how to create appealing layouts.

Assignment 3 – Page Layout

Create responsive page layouts using Flexbox and media queries.

Assignment 4 – Page Layout
